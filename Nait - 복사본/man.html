Nait API Document

Nait이란 Nootloader의 서브 프로젝트로, Nootloader 커뮤니티의 글쓰기 엔진 입니다. Nait이 포함하고 있는 서브 프로젝트는 Wait(nait 백엔드), Nait(nait 프론트엔드) 가 있습니다.

주제 리스트

/dabg

여기로 get 요청 보내면 주제 목록을 반환합니다.

주제 만들기

/dabg
name : (name), discription : (discription)

여기로 post 요청 보내면 그 이름을 가진 주제를 만들어줍니다. 성공하면 "주제가 성공적으로 생성되었습니다."이 반환되고, 주제 이름이 곂치면 "이미 존제하는 주제 이름입니다."이 반환됩니다.

주제 수정

/dabg/(dabgname)/edit
name : (name), discription : (discription)

여기로 post요청하면 그 주제를 수정합니다.

주제 삭제

/dabg/(dabgname)/

여기로 DELETE요청하면 주제 삭제함

영역 리스트

/dabg/(dabgname)

여기로 get요청 보내면 그 주제에 속해있는 영역의 목록을 반환합니다.

영역 생성

/dabg/(dabgname)
name : (name), discription : (discription)

여기로 post요청하면 그 주제에 속해있는 영역을 생성합니다.

영역 수정

/dabg/(dabgname)/(lyoiki)/edit
name : (name), discription : (discription)

여기로 post요청하면 그 주제에 속해있는 영역을 수정합니다.

영역 삭제

/dabg/(dabgname)/(lyoiki)

여기로 DELETE요청하면 영역 삭제함

글 리스트

/dabg/(dabgname)/(lyoiki)

여기로 get요청하면 해당 주제/영역 에 속해있는 글 목록을 반환합니다.

글 생성

/dabg/(dabgname)/(lyoiki)
title : (title), body : (body)

글 수정

/dabg/(dabgname)/(lyoiki)/(docid)
title : (title), body : (body)

여기로 post요청하면 그 주제에/영역/글을 수정합니다.

글 삭제

/dabg/(dabgname)/(lyoiki)/(docid)

여기로 DELETE요청하면 글 삭제함

여기로 post요청하면 해당 주제/영역 에 속해있는 글을 하나 만듭니다.

글 내용

/dabg/(dabgname)/(lyoiki)/(docid)

여기로 get요청하면 해당 글의 글 내용을 불러옵니다

댓글생성

/dabg/(dabgname)/(lyoiki)/(docid)
body : (body)

여기로 post요청하면 해당 주제/영역/글에 속해있는 댓글을 생성합니다.

댓 리스트

/dabg/(dabgname)/(lyoiki)/(docid)/comment

여기로 get요청하면 댓 리스트 뽑아줌
